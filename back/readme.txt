Projeto Spring Boot

Necessario utilizar java 11 para executar o projeto

Execute o MovieApplication
A aplicação vai subir no http://localhost:8080/

Voce pode acessar o h2 atraves da url:
    http://localhost:8080/h2-console
usuario: user
senha: password

A url referente ao intervalo de premios é:
    http://localhost:8080/api/producers/intervalo-de-premios
